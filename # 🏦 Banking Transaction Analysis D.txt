# 🏦 Banking Transaction Analysis Dashboard

## 📌 Project Overview

The Banking Transaction Analysis Dashboard is an end-to-end Data Analytics project developed using **PostgreSQL**, **SQL**, **Power BI**, and **DAX**. The dashboard provides valuable insights into banking transactions, customer behavior, branch performance, account types, and transaction channels to support data-driven decision-making.

---

## 🎯 Business Problem

Banks process thousands of transactions daily across multiple branches and channels. Analyzing this data manually is difficult and time-consuming.

This dashboard helps management:

- Monitor transaction trends
- Analyze customer segments
- Track branch performance
- Evaluate product categories
- Support business decisions using interactive dashboards

## 📈 Key Business Insights

- Transaction volume reached its highest level in **2025**.
- **Web** is the most preferred transaction channel.
- **Retail** is the largest customer segment.
- **Lending** generated the highest transaction value.
- Debit transactions exceeded credit transactions, indicating higher cash outflow.

---

## 💡 Business Recommendations

- Increase investment in digital banking channels.
- Improve low-performing branches.
- Strengthen customer retention in the Retail segment.
- Promote deposit and credit products.
- Monitor KPIs regularly for better decision-making.
