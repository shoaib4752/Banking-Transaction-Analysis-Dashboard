# 🏦 Banking Transaction Analysis Dashboard

An end-to-end Banking Transaction Analysis project developed using **Power BI, SQL, DAX, Power Query, and PostgreSQL**. This project transforms raw banking transaction data into interactive dashboards that help analyze customer activity, transaction trends, branch performance, and account insights to support business decision-making.

---

# 📌 Project Objectives

- Analyze banking transaction performance
- Monitor customer and account activity
- Evaluate branch performance
- Identify transaction trends and patterns
- Build interactive business intelligence dashboards

---

# 🛠 Tools & Technologies

- Power BI
- SQL
- PostgreSQL
- Power Query
- DAX
- CSV Dataset

# 📊 Dashboard Pages

- Total Transactions

- Transaction Channel Analysis

- Top Performing Branches

# 📈 Business Insights

- Deposits contributed the highest transaction value.
- Digital banking channels handled the majority of transactions.
- A few branches generated a significant share of total transaction volume.
- Monthly transactions remained stable with noticeable peak periods.
- Savings accounts represented the largest customer base.
- Top customers contributed significantly to total transaction value.
- Branch performance varied across regions.
- Online banking usage continued to dominate customer transactions.
- Customer activity differed across account types.
- Interactive dashboards enable quick business performance monitoring.
